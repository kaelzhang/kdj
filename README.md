# kdj

FinTech utility method to calculate KDJ.

## Install

```sh
$ npm install kdj
```

## Usage

```js
import kdj from 'kdj'
```

## kdj(close, low, high, )

- **close** `Array.<Number>` array of closing prices.
- **low** `Array.<Number>` array of low prices.
- **high** `Array.<Number>` array of high prices.
